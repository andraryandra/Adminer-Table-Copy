# Cara Install Extension Adminer Table Copy

## Opsi 1: Install dari Firefox Add-ons (Paling Mudah) ✅

1. Buka: https://addons.mozilla.org/
2. Cari "Adminer Table Copy"
3. Klik "Add to Firefox"
4. Selesai! Extension terinstall otomatis

**Keuntungan:**
- ✅ Auto-update
- ✅ Terverifikasi oleh Mozilla
- ✅ Install dengan 1 klik

---

## Opsi 2: Install Manual (Chrome/Edge/Brave)

### Langkah-langkah:

1. **Download Extension**
   ```
   Download file ZIP dari GitHub Releases
   ```

2. **Extract ZIP File**
   ```
   Extract ZIP ke folder (misal: adminer-table-copy)
   ```

3. **Buka Halaman Extensions**
   - **Chrome/Brave**: Buka `chrome://extensions/`
   - **Edge**: Buka `edge://extensions/`

4. **Aktifkan Developer Mode**
   - Toggle "Developer mode" di pojok kanan atas

5. **Load Extension**
   - Klik tombol "Load unpacked"
   - Pilih folder extension yang sudah di-extract
   - Extension akan terinstall

6. **Selesai!**
   - Extension siap digunakan
   - Buka halaman Adminer untuk menggunakan

---

## Opsi 3: Install Manual (Firefox)

1. **Download Extension**
   ```
   Download file ZIP dari GitHub Releases
   ```

2. **Extract ZIP File**
   ```
   Extract ZIP ke folder
   ```

3. **Buka Firefox Debugging**
   - Buka: `about:debugging`
   - Klik "This Firefox"

4. **Load Temporary Add-on**
   - Klik "Load Temporary Add-on"
   - Pilih file `manifest.json` dari folder extension

5. **Selesai!**
   - Extension terinstall (temporary)
   - Catatan: Extension akan hilang setelah restart browser

## Update Extension

### Jika Install dari Firefox Add-ons:
- ✅ Auto-update otomatis
- Tidak perlu melakukan apapun

### Jika Install Manual:
1. Download versi baru dari Releases
2. Extract ke folder yang sama (replace file lama)
3. Buka halaman extensions
4. Klik tombol reload pada extension

---

## Troubleshooting

### Extension tidak muncul setelah install
- ✅ Pastikan Developer mode aktif
- ✅ Refresh halaman extensions
- ✅ Cek apakah ada error di console

### Extension tidak bekerja di Adminer
- ✅ Refresh halaman Adminer
- ✅ Pastikan halaman adalah Adminer (bukan halaman lain)
- ✅ Cek console browser untuk error

### Error saat load extension
- ✅ Pastikan semua file ada di folder
- ✅ Pastikan `manifest.json` valid
- ✅ Pastikan semua icon ada (16, 48, 128)

### Extension hilang setelah restart (Firefox)
- ✅ Ini normal untuk temporary add-on
- ✅ Install ulang dengan cara yang sama
- ✅ Atau install dari Firefox Add-ons untuk permanen
